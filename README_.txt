--------------------------------------- Projet C : Dessin Vectoriel-----------------------------------------------------------------

Ce programme C est un logiciel de dessin vectoriel. Il permet d'ajouter différentes formes sur des calques puis de les dessiner.


-----------------------------------------------------------------Auteurs -----------------------------------------------------------

- ASSOUAD Adrien (L1 groupe F)
- CHAN PENG Julien (L1 groupe F)

------------------------------------------------------ Installation et exécution ---------------------------------------------------

- Utiliser un logiciel ou terminal qui permet de compiler le programme.
- Si vous utilisez un logiciel de programmation C : ouvrir le fichier main et exécutez le.



----------------------------------------------------------Les fonctionnalités ------------------------------------------------------


- L'utilisateur peut créer des points, des lignes, des cercles, des carrés, des rectangles et des polygones.
- Pour créer ses formes l'utilisateur doit rentrer les informations des formes.
-Il peut les afficher.
-Il peut les supprimer.


------------------------------------------------ Les bugs connus et informations importantes ---------------------------------------

- Il n'y a pas de saisie sécurisée donc si les valeurs rentrées ne sont pas bonnes cela nous sort du programme.
- Pas de bugs connus
-Si ordinateur sous windows modifier commande clear (system("clear") par system("cls") ).
